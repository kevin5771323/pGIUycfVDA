# 项目简介

## 核心功能
- 基于 YOLOv8 的地质雷达检测系统
- 提供 1100 张标注数据集，包含 ['parabol'] 类别
- 支持图片、视频及摄像头进行检测
- 支持选择模型并自定义置信度、IOU 阈值
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 提供选择显示标签和原图的功能

## 适用场景
- 地质雷达图像分析
- 目标检测与识别研究
- 计算机视觉应用开发
- 模型训练与评估
- 批量图像处理

## 技术特点
- 使用 Pycharm+Anaconda 或 VSCode+Anaconda 开发环境
- 基于 Python 3.9，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等技术栈
- 配置成功环境可直接使用，无需额外训练
- 提供完整的 UI 界面源文件、图标和项目运行教程
- 支持单张图片、图片文件夹、视频文件和本地摄像头的检测方式
